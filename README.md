Библиотека линейной алгебры
Объектно-ориентированная библиотека для операций линейной алгебры на языке Java. Библиотека предоставляет классы для работы с векторами (2D, 3D, 4D) и матрицами (3x3, 4x4).

Структура проекта

        Vector3.java
        Vector4.java
    matrix/
        Matrix3x3.java
        Matrix4x4.java
